== Description ==

**Important Note:**
<p>For Change Ordering Of CSS AND Js On Particular Page Wise You Need To First Create Page Then Open that page on Front end first time.  </p>

**Features:**
You Can Change Css And Js Ordering Page wise.


== Installation ==

1. Upload `css-js-ordeing-page-wise` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Create New Page From Page Option
1. Then Reload That Page Once From Front Side
1. Then You Can Change Css And Js Ordering from Edit Page Side
